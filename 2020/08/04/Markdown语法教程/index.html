关于Markdown语法的学习

标题

  • H1~H6大纲标题
    【语法】使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推.
    【示例】

段落格式

  • 换行
    【语法】段落的换行是使用两个以上空格加上回车
    【示例】 harryborrsion.cn
    harryborrsion.cn
  • 分段
    【语法】在段落最后使用一个空行来分段
    【示例】 harryborrsion.cn

harryborrsion.cn

  • 字体 【示例】
  • 斜体文本*
    斜体文本
  • 粗体文本*
    粗体文本
  • 粗斜体文本*
    粗斜体文本
  • 分割线
    【语法】一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西
    【示例】
  • 删除线
    【语法】只需要在文字的两端加上两个波浪线 ~~
    【示例】 harryborrsion.cn
    harryborrsion.cn
  • 下划线
    【语法】可以通过 HTML 的<u>标签来实现
    【示例】
    <u>带下划线文本</u>
    带下划线文本

列表

  • 无序列表
    【语法】使用星号(*)、加号(+)或是减号(-)作为列表标记，在标记后面要添加一个空格，然后再填写内容
    【示例】
  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项
  • 第一项
  • 第二项
  • 第三项
  • 有序列表
    【语法】有序列表使用数字并加上 . 号来表示。
    【示例】
  1. 第一项
  2. 第二项
  3. 第三项
  • 列表嵌套
    【语法】列表嵌套只需在子列表中的选项前面添加四个空格。
    【示例】
  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素

【语法】

感叹号+方括号+普通括号
方括号里是图片代替的文字
普通括号是图片的地址

表格

【语法】Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。
对齐方式

我们可以设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐。
  • :- 设置内容和标题栏居左对齐。
  • :-: 设置内容和标题栏居中对齐。
    【示例】
表头 表头 表头
单元格 单元格 单元格
单元格 单元格 单元格
